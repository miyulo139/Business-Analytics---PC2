# Business Analytics — PC2: Spotify Perú

Repositorio del proyecto del curso de Business Analytics (PC2) que analiza datos de Spotify en Perú.
Consta de un proyecto de análisis y visualización sobre el consumo de música en Spotify en Perú. Resuelve preguntas sobre artistas, géneros y tendencias temporales a partir de datos preparados y proporciona resultados reproducibles (notebook), documentación metodológica (informe) y materiales para un dashboard interactivo (Power BI).

## Contenido del repositorio
- `BA-Presentación_Spotify.pdf` — Presentación (diapositivas) del proyecto.
- `Informe PC2 - BA - Spotify.pdf` — Informe final con metodología, análisis y conclusiones.
- `Datos_Dashboard_PowerBI.zip` — Conjunto de datos y archivos para el dashboard en Power BI.
- `Spotify_Peru_BA_PC2 (2).ipynb` — Notebook Jupyter con el análisis reproducible (Python).

## Resumen
El objetivo del proyecto es explorar y visualizar información sobre canciones, artistas y métricas de consumo en Spotify en Perú. El análisis incluye:

- Recolección y limpieza de datos.
- Análisis descriptivo de popularidad, géneros y artistas.
- Identificación de tendencias temporales y patrones de consumo.
- Visualizaciones para apoyar un dashboard en Power BI.
- Recomendaciones basadas en los hallazgos.

## Reproducir el análisis
1. Clona el repositorio:
   ```bash
   git clone https://github.com/miyulo139/Business-Analytics---PC2.git
   ```
2. Abre el notebook `Spotify_Peru_BA_PC2 (2).ipynb` en JupyterLab o Jupyter Notebook.
3. Instala las dependencias necesarias (por ejemplo):
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```
4. Si usas Power BI, descomprime `Datos_Dashboard_PowerBI.zip` y carga los archivos en Power BI.

## Dependencias
- Python 3.8+ (recomendado)
- pandas
- numpy
- matplotlib
- seaborn
- jupyter / jupyterlab
